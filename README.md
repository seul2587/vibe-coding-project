1. 프로젝트 제목: 나만의 전공 용어집

이 프로젝트는 HTML, CSS, JavaScript(ES6+)와 브라우저의 localStorage를 활용하여 만든 '나만의 전공 용어집' 웹 애플리케이션입니다.

백엔드 서버 없이 단일 HTML 파일로 작동하며, 사용자가 직접 전공 용어와 '나만의 정의', 이미지, 카테고리를 저장하고 관리할 수 있는 개인 맞춤형 지식 관리 시스템(PKMS)입니다.

2. 사용자 및 제공 가치

"사용자는 누구인가?"

1순위 사용자 (Primary User):
비전공자 출신으로 AI/ML 및 공학 분야 대학원에 막 입학한 '나 자신'.

확장 사용자 (Secondary Users):

AI/ML 및 기타 기술 분야를 독학하는 모든 입문자.

새로운 전공 분야의 지식을 체계화하고 싶은 모든 학생.

"어떤 가치를 제공하는가?"

학습 효율 극대화:
'그래서 CNN이랑 RNN의 결정적 차이가 뭐지?' 같이 '나만의 언어'로 정리된 정의를 저장하고 언제든 다시 찾아볼 수 있어, 반복적인 구글링 시간을 낭비하지 않고 학습 효율을 극대화합니다.

개인 맞춤형 지식창고 (PKMS):

이미지 첨부: 논문 다이어그램, 강의 중 스크린샷, 손으로 그린 그림 등을 함께 저장하여 텍스트만으로는 부족한 시각적 이해를 돕습니다.

자유로운 카테고리: 'ML', 'DL', 'Python', '통계', '논문별' 등 원하는 대로 카테고리를 생성하고 분류하여 지식을 체계화할 수 있습니다.

3. 주요 기능

이 프로젝트는 웹의 핵심 기술만을 사용하여 CRUD(Create, Read, Update, Delete) 기능을 localStorage 기반으로 구현했습니다.

① 용어 추가 (Create)

전공 용어와 **'나만의 정의'**를 입력합니다.

카테고리 선택 및 생성:

기존에 만들었던 카테고리를 드롭다운에서 선택할 수 있습니다.

[ + 새 카테고리 추가 ] 옵션을 통해 원하는 이름의 새 카테고리를 즉시 생성할 수 있습니다.

이미지 첨부 (2가지 방식):

이미지 URL을 직접 붙여넣기 할 수 있습니다.

로컬 파일 업로드: 내 컴퓨터의 이미지 파일(png, jpg 등)을 업로드하면, 이미지가 Base64 (Data URL) 텍스트로 자동 변환되어 localStorage에 함께 저장됩니다. (서버 불필요)

참고: localStorage 용량 보호를 위해 2MB 초과 파일은 업로드가 제한됩니다.

② 목록 조회 및 필터링 (Read)

저장된 모든 용어 목록을 최신순으로 조회합니다.

실시간 검색: 검색창에 키워드를 입력하면, '용어'나 '의미'에 해당 키워드가 포함된 항목만 실시간으로 필터링됩니다.

카테고리 필터: 특정 카테고리를 선택하여 해당 용어만 모아볼 수 있습니다.

③ 용어 수정 (Update)

'수정' 버튼을 누르면 해당 항목의 내용(용어, 정의, 카테고리, 이미지)이 상단 폼으로 로드됩니다.

자유롭게 내용을 수정한 뒤 '변경사항 저장'을 누르면 즉시 반영됩니다.

④ 용어 삭제 (Delete)

'삭제' 버튼을 누르면 해당 항목이 목록과 localStorage에서 즉시 영구 삭제됩니다.

4. 실행 방법

이 프로젝트는 별도의 설치나 빌드 과정이 필요 없습니다.

my_glossary.html (또는 원하는 이름의 .html 파일)을 다운로드합니다.

해당 파일을 웹 브라우저(크롬, 엣지, 파이어폭스 등)로 열기만 하면 바로 사용할 수 있습니다.

주소창에 file:///.../my_glossary.html과 같이 표시됩니다.

5. 사용된 기술

HTML5: 시맨틱 마크업

CSS3 (Tailwind CSS): CDN을 활용한 반응형 UI/UX 디자인

JavaScript (ES6+):

localStorage를 활용한 데이터 영속성 처리

DOM 조작을 통한 동적 UI 구현

이벤트 처리 (폼 제출, 수정, 삭제, 실시간 검색/필터링)

FileReader API를 활용한 로컬 이미지 Base64 인코딩
